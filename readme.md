## mocha demo

clone the repository
```bash
$ git clone https://github.com/sharan01/mochademo.git
```
####install dependencies
```bash
$ cd mochademo
$ npm install
```
#### run tests
```bash
$ npm test
```
#### run server
```bash
$ npm run dev
```
